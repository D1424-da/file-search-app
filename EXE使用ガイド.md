# ファイル検索アプリ - EXE版 使用ガイド

## 🎉 ビルド完了

EXEファイルが正常に作成されました！

## 📁 ファイルの場所

```
dist\ファイル検索アプリ.exe
```

## 🚀 実行方法

### 方法1: 単独で実行
`dist\ファイル検索アプリ.exe` をダブルクリックするだけで起動します。

### 方法2: 配布用に準備
他のPCで使用する場合は、以下のフォルダ構成で配布してください：

```
配布フォルダ/
  ├── ファイル検索アプリ.exe
  ├── config/
  │   └── default_settings.json
  └── data_storage/  (自動作成されます)
```

## ⚠️ 重要な注意事項

### 1. ウイルス対策ソフトの警告
初回起動時にWindowsやウイルス対策ソフトから警告が出る場合があります。これは、PyInstallerで作成されたEXEファイルが未署名であるためです。

**対処法:**
- Windows Defender: 「詳細情報」→「実行」をクリック
- セキュリティソフト: 例外として登録

### 2. OCR機能について
TIF画像のOCR機能を使用する場合、別途Tesseract-OCRのインストールが必要です。

**インストール方法:**
1. https://github.com/UB-Mannheim/tesseract/wiki からダウンロード
2. インストール後、アプリの設定でTesseractのパスを指定

### 3. ファイルサイズ
生成されたEXEファイルは約100-150MBになります。これは正常です。

## 📋 システム要件

- OS: Windows 10/11 (64bit)
- メモリ: 4GB以上推奨
- ディスク空き容量: 500MB以上

## 🔧 トラブルシューティング

### 起動しない場合
1. **管理者として実行**: EXEファイルを右クリック→「管理者として実行」
2. **パスの確認**: configフォルダが同じ場所にあることを確認
3. **Windows Defenderの除外設定**: 実行フォルダを除外リストに追加

### エラーが表示される場合
1. data_storageフォルダへの書き込み権限を確認
2. 他のセキュリティソフトがブロックしていないか確認
3. 元のPythonスクリプトが正常に動作するか確認

## 🔄 再ビルドする場合

アプリを更新して再度EXEファイルを作成する場合:

```batch
build_exe.bat
```

を実行するだけです。
